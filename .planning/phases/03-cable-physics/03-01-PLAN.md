# Plan 03-01: live aic_eval scoring snapshot + cable USD topology probe

## Objective

Two-pronged infra plan that lands the source-of-truth artifacts the rest of Phase 3 depends on:

1. **Live `/scoring/*` snapshot** — capture exact topic shapes from `aic_eval` Docker container (frame names, message payloads, publish rates). PARITY-07/08 + SCENE-06 cannot author the right surface without these.
2. **Cable USD topology probe** — walk the deactivated cable subtree, inventory all RigidBody links, joint chain, current mass/inertia. SCENE-05 cable physics authoring needs this.

## Tasks

### Task 1 — Bring up aic_eval, capture /scoring/* + /tf

Author `exts/aic-dt/scripts/snapshot_aic_eval_scoring.sh`:

```bash
#!/bin/bash
# Bring up aic_eval headless + CheatCode for ~30s, capture /scoring/* and /tf
cd ~/Documents/aic
./scripts/run_cheatcode.sh headless &
WRAPPER_PID=$!
sleep 30  # let containers stabilize + CheatCode start moving

OUTDIR=/home/aaugus11/Documents/isaac-sim-mcp/.planning/phases/03-cable-physics/snapshot
mkdir -p "$OUTDIR"

docker exec aic_eval bash -c '
source /opt/ros/kilted/setup.bash
echo "=== topic list ==="
ros2 topic list
echo
echo "=== /scoring/insertion_event info + 5 echoes ==="
ros2 topic info /scoring/insertion_event -v
timeout 10 ros2 topic echo /scoring/insertion_event --once 2>&1 | head -20
echo
echo "=== /scoring/tf info + 1 echo ==="
ros2 topic info /scoring/tf -v
timeout 5 ros2 topic echo /scoring/tf --once 2>&1 | head -60
echo
echo "=== /tf frames containing cable/plug/port ==="
ros2 run tf2_tools view_frames -f /tmp/frames 2>&1 | head -10
ros2 topic echo /tf --once 2>&1 | head -200
' > "$OUTDIR/aic_eval_scoring_capture.txt" 2>&1

# Tear down
./scripts/run_cheatcode.sh stop
wait $WRAPPER_PID 2>/dev/null
echo "Snapshot saved to $OUTDIR/aic_eval_scoring_capture.txt"
```

Make executable; run; tee output to `.planning/phases/03-cable-physics/snapshot/aic_eval_scoring_capture.txt`.
