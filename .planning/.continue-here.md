## Critical Anti-Patterns

| Pattern | Description | Severity | Mitigation |
|---------|-------------|----------|------------|
| Trusting metadata over rendered output | `get_active_viewport()` reported wrist but actual frames came from Persp. Spent time on pose math when the bug was wrong-camera-entirely. | advisory | Verify image stream matches expectation before pose-math analysis. Capture topic, inspect visually. |
| Hardcoded constants in "verified" upstream code | `MARKER_SIZE = 0.032` was hardcoded for JETANK. Origin's so_arm101 path was half-finished — config block existed, hardcodes weren't parameterized. | advisory | When using upstream code "verified for X" on robot Y, audit hardcoded constants. "Verified" ≠ "parameterized for all robots." |
| Parallel clones of the same repo with divergent uncommitted edits | Two clones of aruco_camera_localizer (`~/Projects/RoboSort/...` vs `~/Desktop/ros2_ws/src/...`). bashrc sourced Desktop. Edits to Projects/RoboSort silently ignored at runtime. | advisory | Run `python3 -c 'import pkg; print(pkg.__file__)'` to identify which clone is on PYTHONPATH BEFORE editing. Consolidate to one when possible. |
