# Lego block assets

Place the lego USD files here. The extension will use them instead of Nucleus when this folder exists.

**Expected files:**

| Color | Files |
|-------|-------|
| Red   | `lego_red_2x2.usd`, `lego_red_2x3.usd`, `lego_red_2x4.usd` |
| Green | `lego_green_2x2.usd`, `lego_green_2x3.usd`, `lego_green_2x4.usd` |
| Blue  | `lego_blue_2x2.usd`, `lego_blue_2x3.usd`, `lego_blue_2x4.usd` |

Download from your Nucleus `omniverse://localhost/Projects/so-arm101/legos/` and copy them here.
